# Banana Daydream Slider

A custom animated slider for the Ellianos Banana Daydream promotional campaign featuring smooth crossfade transitions, drifting cloud animations, and responsive design.

## Features

- Auto-rotating slides with 5-second intervals
- Smooth crossfade transitions
- Animated drifting cloud background
- Dot navigation
- Touch/swipe support for mobile
- Fully responsive (desktop & mobile)

## Files

- `index.html` — Production version using remote WordPress URLs
- `index-2.html` — Local development version using `assets/` directory
- `assets/` — Local image assets (slide images, cloud layers)

## Usage

Embed the slider by copying the contents of `index.html` into any WordPress page or HTML document. The slider is self-contained with inline styles and scripts.
